# Trie

- a tree data structure that is used to store a set of strings
    - root note represents empty string
    - outgoing edges are associated with a label (e.g. a label)

- Applications
    - storing a dynamic set of strings
    - sort strings
    - Autocompletion
    - Spell-checking
